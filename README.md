# community-service
💬 MERN Stack 기반 커뮤니티 서비스
## 1. 기본 기능 구성
- 사용자 등록, 로그인, 로그아웃 (/)
- 소개 페이지 (/about)
- 게시판 (/board)
  - 제목, 작성자, 작성일, 본문 포함
  - 읽기, 쓰기, 수정, 삭제 기능
- 갤러리 (/gallery)
  - 제목, 작성자, 작성일, 사진 포함
  - 읽기, 쓰기, 수정, 삭제 기능
## 2. 개발 환경
- Front-end : React
- Back-end : Node.js
- Database : MongoDB
